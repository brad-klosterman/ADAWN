# Spider Audio/CV Merger & Splitter

## Overview
The Spider Audio/CV is a dual-purpose utility device that combines audio and CV signal routing in a single unit. It features both an audio merger/splitter section and a CV merger/splitter section, making it essential for complex routing scenarios and modular-style patching.

## Features
- Audio merger (4-to-1)
- Audio splitter (1-to-4)
- CV merger (4-to-1)
- CV splitter (1-to-4)
- Independent audio/CV sections
- Automatic signal scaling
- Compact design
- Flexible routing options

## Core Sections

### Audio Merger
- 4 stereo inputs
- 1 stereo output
- Automatic mixing
- Signal summation
- Level preservation

### Audio Splitter
- 1 stereo input
- 4 stereo outputs
- Clean signal distribution
- Identical copies
- No signal degradation

### CV Merger
- 4 CV inputs
- 1 CV output
- Signal summation
- Voltage averaging
- Bipolar handling

### CV Splitter
- 1 CV input
- 4 CV outputs
- Identical distribution
- Voltage preservation
- No signal loss

## Connections

### Audio Section
- 4 stereo merger inputs
- 1 stereo merger output
- 1 stereo splitter input
- 4 stereo splitter outputs

### CV Section
- 4 CV merger inputs
- 1 CV merger output
- 1 CV splitter input
- 4 CV splitter outputs
