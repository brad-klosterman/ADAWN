# PH-90 Phaser

## Overview
The PH-90 Phaser is a classic phase shifting effect that creates sweeping, organic modulation through frequency notches. It features an LFO for automatic modulation and CV control for creative applications.

## Features (pp. 806-807)
- Classic phaser algorithm
- LFO with multiple waveforms
- Feedback control
- Rate and depth control
- CV modulation options
- Stereo processing

## Parameters

### Rate
- Range: 0 to 127
- Controls LFO speed
- Tempo sync option
- CV controllable

### Depth
- Range: 0 to 127
- Modulation amount
- Controls sweep width
- CV modulation input

### Feedback
- Range: 0 to 127
- Resonance control
- Enhances effect intensity
- CV controllable

### Frequency
- Base frequency setting
- Range: 0 to 127
- Sets center frequency
- CV modulation input

### LFO Section
- Multiple waveforms:
  - Triangle
  - Sine
  - Square
  - Random
- LED rate indicator

## Connections

### Audio
- Input Left/Right
- Output Left/Right
- True stereo processing

### CV
- Rate CV In
- Depth CV In
- Feedback CV In
- Frequency CV In
