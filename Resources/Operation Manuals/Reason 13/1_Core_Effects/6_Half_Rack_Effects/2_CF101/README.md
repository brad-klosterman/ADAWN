# CF-101 Chorus/Flanger

## Overview
The CF-101 is a combined chorus and flanger effect that can create everything from subtle stereo widening to extreme modulation effects. It features an LFO with multiple waveforms and manual or tempo-synced operation.

## Features (pp. 793-794)
- Combined chorus/flanger
- Multiple LFO waveforms
- Tempo sync capability
- Stereo processing
- CV/Gate modulation
- Feedback control

## Parameters

### Rate
- Manual: 0.1 Hz to 10 Hz
- Tempo sync option
- CV control input
- LED indicator

### Depth
- Range: 0 to 127
- Controls modulation amount
- CV modulation input
- Affects stereo width

### Feedback
- Range: -64 to 63
- Positive/negative feedback
- Resonance control
- CV modulation input

### Delay
- Sets base delay time
- Affects character (chorus/flanger)
- Range: 0 to 127
- CV controllable

### LFO Waveform
- Sine wave
- Triangle wave
- Sawtooth wave
- Square wave

## Connections

### Audio
- Input Left/Right
- Output Left/Right

### CV
- Rate CV In
- Depth CV In
- Feedback CV In
- Delay CV In
