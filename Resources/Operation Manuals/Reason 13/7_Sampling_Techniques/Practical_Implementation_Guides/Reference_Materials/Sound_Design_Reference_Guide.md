# Sound Design Reference Guide

## Hybrid Drum Design

### 1. Kick Drum References
```
Modern Electronic:
□ Jon Hopkins - "Open Eye Signal"
  - Focus: Sub weight + transient detail
  - Key elements: Clean sub, crisp attack
  - Processing: Minimal, precise layering

□ Bonobo - "Kerala"
  - Focus: Organic + electronic fusion
  - Key elements: Warm body, textured attack
  - Processing: Natural compression, subtle saturation

□ Four Tet - "Planet"
  - Focus: Punchy mid-range
  - Key elements: Tight envelope, focused energy
  - Processing: Careful EQ, controlled dynamics
```

### 2. Layer Analysis
```
Sub Layer Characteristics:
□ Electronic Music
  - 40-60Hz: Foundation
  - 60-80Hz: Power
  - 80-100Hz: Translation

□ Pop/Rock
  - 50-70Hz: Weight
  - 70-90Hz: Punch
  - 90-110Hz: Body

□ Hip-Hop/Trap
  - 30-50Hz: Ultra-low
  - 50-70Hz: Impact
  - 70-90Hz: Presence
```

```
□ Glitch Textures

  - Variable density
  - High spread

□ Rhythmic Textures
  - Grain size: 20-50ms
  - Synced density
  - Patterned spread
  - LFO modulation
```

```

□ Characteristics

Mid Range (300Hz-3kHz):
□ Characteristics
  - Presence
  - Definition

High Mids (3-8kHz):
□ Characteristics

  - Clarity

High End (8kHz+):
```


## Effect Reference

### 1. Time-Based Effects
```
Reverb Characteristics:
□ Room
  - Size: 0.2-1.0s
  - Early reflections
  - Tight diffusion

□ Hall
  - Size: 1.5-4.0s
  - Rich tail
  - Wide diffusion

□ Ambient
  - Size: 3.0-10.0s
  - Modulated tail
  - Complex diffusion

Delay Types:
□ Rhythmic
  - Time: 1/8, 1/16
  - Moderate feedback
  - Filtered repeats

□ Spatial
  - Time: 30-100ms
  - Low feedback
  - Wide stereo

□ Creative
  - Variable time
  - High feedback
  - Heavy processing
```

### 2. Dynamic Processing
```
Compression Styles:
□ Subtle
  - Ratio: 2:1-4:1
  - Soft knee
  - Moderate attack/release

□ Punchy
  - Ratio: 4:1-8:1
  - Hard knee
  - Fast attack, medium release

□ Extreme
  - Ratio: 10:1+
  - Limited ceiling
  - Character coloration
```

## Production Tips

### 1. Monitoring Reference
```
Level Guidelines:
□ Full Mix
  - Peak: -6dB
  - RMS: -14dB
  - Dynamic range: 8-12dB

□ Individual Elements
  - Kicks: -12dB peak
  - Snares: -14dB peak
  - Pads: -18dB RMS
```

### 2. Processing Order
```
Signal Flow Options:
□ Clean Chain
  1. EQ (subtractive)
  2. Compression
  3. EQ (additive)
  4. Effects

□ Creative Chain
  1. Distortion/Character
  2. Filter/EQ
  3. Modulation
  4. Space/Time effects
```
