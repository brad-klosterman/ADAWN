## Creative Effects Usage

### 1. Time-Based Effects
```
Reverb Applications:
□ Ambient Space
  - Pre-delay: 20-50ms
  - Decay: 3.0s+
  - High diffusion
  Example: Brian Eno "Thursday Afternoon"
  - Long, evolving tails
  - Modulated space
  - Filtered returns

□ Room Character
  - Pre-delay: 5-15ms
  - Decay: 0.8-1.2s
  - Early reflections
  Example: Bonobo "Black Sands"
  - Natural space
  - Room definition
  - Depth creation

Delay Usage:
□ Rhythmic Elements
  - Synced times
  - Filtered feedback
  - Pattern creation
  Example: Four Tet "Angel Echoes"
  - Delay patterns
  - Feedback control
  - Effect automation
```

### 2. Dynamic Processing
```
Compression Styles:
□ Subtle Glue

  - Soft knee

  Example: Tycho "Awake"

□ Punchy Impact

  Example: Chemical Brothers "Galvanize"
  - Aggressive punch

□ Creative Crush

```

## Technical Guidelines

### 1. Mix Approach
```
Level Management:
□ Dynamic Range
  - Peaks: -6dB
  - Average: -14dB
  - Sub bass: -12dB
  Example: Jon Hopkins "Open Eye Signal"
  - Clean headroom
  - Punchy dynamics
  - Clear separation

□ Frequency Balance
  - Sub control
  - Mid presence
  - Air/space
  Example: Tycho "Division"
  - Balanced spectrum
  - Clear elements
  - Open space
```

### 2. Processing Order
```
Signal Flow:
□ Clean Chain
  1. Corrective EQ
  2. Dynamic control
  3. Tonal shaping
  4. Effects/space
  Example: Bonobo "Migration"
  - Natural sound
  - Careful processing
  - Spatial depth

□ Creative Chain
  1. Character/drive
  2. Heavy processing
  3. Resampling
  4. Final shaping
  Example: Amon Tobin "ISAM"
  - Heavy processing
  - Multiple passes
  - Complex automation
```
