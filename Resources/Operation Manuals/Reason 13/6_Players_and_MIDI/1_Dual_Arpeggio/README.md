# Dual Arpeggio Player

## Overview
The Dual Arpeggio Player is a sophisticated MIDI effect that combines two independent arpeggiators with extensive pattern generation and performance capabilities. It allows for complex rhythmic and melodic patterns through the interaction of its two arpeggiator engines.

## Features
- Dual arpeggiator engines
- Independent patterns
- Multiple play modes
- Note range control
- Pattern generation
- Velocity control
- Gate length adjustment
- Shuffle settings
- Pattern storage
