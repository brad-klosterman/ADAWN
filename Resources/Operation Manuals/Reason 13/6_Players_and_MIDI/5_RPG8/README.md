# RPG-8 Monophonic Arpeggiator

## Overview
The RPG-8 is a sophisticated arpeggiator that transforms held notes into rhythmic patterns. It features extensive pattern control, velocity modulation, and advanced note processing capabilities, making it an essential tool for creating dynamic musical phrases and evolving sequences.

## Features
- Multiple arpeggio modes
- Pattern generation
- Velocity modulation
- Note length control
- Octave range settings
- Insert mode
- Gate length control
- Shuffle functionality
- MIDI sync capabilities
- CV/Gate outputs
