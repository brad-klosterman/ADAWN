# Scales & Chords Player

## Overview
The Scales & Chords Player is a powerful MIDI processor that combines scale correction with chord generation capabilities. It ensures notes stay within a selected scale while providing automatic chord generation and voicing options.

## Features

- Chord generation
- Multiple scale types
- Chord memory
