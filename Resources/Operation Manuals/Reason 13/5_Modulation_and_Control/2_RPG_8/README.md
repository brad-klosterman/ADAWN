# RPG-8 Monophonic Arpeggiator

## Overview
The RPG-8 is a sophisticated arpeggiator that converts held notes into rhythmic patterns. It features multiple arpeggio modes, pattern generation, velocity control, and extensive MIDI/CV connectivity for deep integration with instruments and modular setups.

## Features
- Multiple arpeggio modes
- Pattern generation
- Note length control
- Velocity modulation
- Octave range settings
- Insert mode
- MIDI/CV outputs
- Sync capabilities
- Real-time control
