# Matrix Pattern Sequencer

## Overview
The Matrix Pattern Sequencer is a powerful modulation tool that combines pattern sequencing with CV/Gate outputs. It features up to 32 patterns of 32 steps each, with independent curve and gate rows for complex modulation and rhythmic control.

## Features
- 32 pattern memory
- 32 steps per pattern
- Curve/Gate outputs
- Pattern chaining
- Resolution control
- Pattern muting
- Real-time recording
- Step programming
- Pattern automation
