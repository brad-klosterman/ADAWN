# Pulsar Dual LFO

## Overview
The Pulsar Dual LFO is a sophisticated modulation generator featuring two LFOs with extensive sync and modulation capabilities. It combines traditional LFO functionality with advanced features like envelope triggering, phase control, and waveform mixing.

## Features
- Dual independent LFOs
- Multiple waveforms
- Sync capabilities
- Level control
- Phase adjustment
- Envelope triggering
- Waveform mixing
- CV modulation
- Rate multiplication
